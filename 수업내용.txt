OSPF 메트릭 구하는 방법 = 출발지에서 목적지까지의 모든 메트릭값을 더한다 => COST (Bandwidth값의 기초한다)
Bandwidth = 대역폭 라고 합니다
대역폭은 넓을수록 값이 증가할수록 빠릅니다 = 메트릭값은 낮아지겠죠?
1차선 보다 10차선이 빠릅니다 < 10차선 보다 100차선이 빠르겠죠?
실제 대역폭 보는 명령어 = show interface serial 0/0 or show interface fastethernet 0/0

s0/0	BW 1544 Kbit		1,544,000
f0/0	BW 100000 Kbit		100,000,000

기준 대역폭 = 10^8= 1억

메트릭 구하는 공식 = 기준 대역폭 / 실제 대역폭
WAN 	100,000,000 / 1,544,000 = 64
LAN	100,000,000 / 100,000,000 = 1
